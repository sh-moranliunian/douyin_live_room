# 关键点
1. 需要在脚本执行的操作系统里提前安装好 `ffmpeg`。
2. 每次拉流之前先随机产生一个匿名cookie。
3. 拉流的 HLS 地址是从直播web页面中提取出来的，位于 `script` 标签内。
